# Keyword-RAG

> **Domain-Specific Retrieval-Augmented Generation for Manufacturing Technical Documents**

Keyword-RAG is a domain-specific Retrieval-Augmented Generation (RAG) framework designed to improve question-answering performance on manufacturing technical documents. The framework enhances retrieval accuracy by augmenting document chunks with representative domain-specific keywords before indexing.

## 💡 Motivation

Conventional RAG suffers from contextual fragmentation when long technical documents are split into fixed-size chunks.

Keyword-RAG improves retrieval by automatically extracting representative keywords and inserting them into document chunks before indexing.
